Подробное описание

Объект для общения с устройствами по последовательному интерфейсу

Объект содержит в себе конструкторы для инициализации подключения и функции для отправки и чтения данных типа byte_t и u_byte_t

Конструктор, который создаёт подключение к устройству со следующими параметрами:

Конструктор, который создаёт подключение к устройству с параметрами, которые передаются в структуре типа port_config.

Аргументы
[in]addressАдрес устройства, к которому требуется подключиться
[in]configСтруктура, содержащая в себе настройки для подключения

Пример:

port_config config;
config.baud_rate = BAUD_115200;
config.byte_size = 8;
config.stop_bits = ONESTOPBIT;
config.parity = NOPARITY;
Serial data_bus("COM1", config);
