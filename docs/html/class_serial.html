Подробное описание

Объект для общения с устройствами по последовательному интерфейсу

Объект содержит в себе конструкторы для инициализации подключения и функции для отправки и чтения данных типа byte_t и u_byte_t

Конструктор с одним аргументом - портом устройства.

Создаёт подключение к устройству со следующими параметрами:

  • baud_rate = 115200
  • byte_size = 8
  • stop_bits = ONESTOPBIT
  • parity = NOPARITY
Аргументы
[in]addressАдрес устройства, к которому требуется подключиться

Пример

Serial data_bus("COM1");

Конструктор с двумя аргументами - портом устройства и настройками соединения.

Создаёт подключение к устройству с параметрами, которые передаются в структуре типа port_config.

Аргументы
[in]addressАдрес устройства, к которому требуется подключиться
[in]configСтруктура, содержащая в себе настройки для подключения

Пример

port_config config;
config.baud_rate = BAUD_115200;
config.byte_size = 8;
config.stop_bits = ONESTOPBIT;
config.parity = NOPARITY;
Serial data_bus("COM1", config);
